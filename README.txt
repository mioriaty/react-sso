
1. Định nghĩa SSO
- Là cơ chế cho phép user có thể truy cập nhiều trang web, ứng dụng mà chỉ cần đăng nhập 1 lần.
- Một khi đã được định danh ở một website A, thì cũng được định danh tương tự ở website B mà không cần đăng nhập lại.

1.1 Ưu và nhược điểm
- Ưu điểm:
  + Giảm số lượng username và password mà user phải nhớ.
  + Giảm thời gian đăng nhập.
  + Tăng tính bảo mật.
- Nhược điểm:
  + Nếu bị hack, hacker có thể truy cập vào nhiều website mà không cần đăng nhập.
  + Nếu user quên logout, người khác có thể truy cập vào tài khoản của user.

2. Auth0
- Auth0 là một dịch vụ quản lý đăng nhập và xác thực người dùng.
- Auth0 cung cấp các API để xác thực người dùng thông qua các dịch vụ khác như Google, Facebook, Twitter, Github, ...
- Auth0 cung cấp các SDK để xây dựng ứng dụng trên nhiều nền tảng như Web, Mobile, Desktop, ...
